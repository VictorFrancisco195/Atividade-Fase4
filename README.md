# Atividade - Muito estilo sem sofrimento e Rede Social dos Desenvolvedores
Projeto desenvolvido para a atividade avaliativa "Muito estilo sem sofrimento e Rede Social dos Desenvolvedores" da FIAP, do curso de Sistemas de Análise e Desenvolvimento (ADS).

## Sobre a atividade
 
A proposta consistia em recriar, em HTML, CSS e Tailwind CSS, uma das telas previamente desenhadas na Fase 2 do projeto Fintech, e publicar o resultado em um repositório público no GitHub.
 
## Tela desenvolvida
 
**Dashboard** — tela inicial da aplicação fintech, com:
 
- Cards de resumo (Saldo Total, Receitas, Despesas, Investimentos)
- Gráfico de barras comparando Receitas vs Despesas
- Gráfico de pizza com despesas por categoria
- Lista das últimas transações
- Menu lateral de navegação
- Layout responsivo para dispositivos mobile

## Tecnologias utilizadas
 
- **HTML5**
- **CSS3**
- **Tailwind CSS**
- **JavaScript (menu responsivo mobile)**
